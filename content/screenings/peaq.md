## Product

EVM-compat layer 1 for DePINs (the "Machine Economy").

Can use their SDK to build your own DePIN network. For example machine IDs,
access control, payment processing, data storage, etc. Available in JS and Python.

Built on Polkadot, using some parachain thing. Set to handle 100,000+ TPS. Fully
open sourced, on GitHub.

Native DEX "MACHINEX".

Website looks good. Marketing on point. Browsing through the documentation it
seems good.

Raised $41M in total. First in mid 2022. Latest in late October 2025. (Rootdata).

## Token

Token $PEAQ. Used to pay transaction fees, stake to validate. Also governance,
with plans to gradually decentralize decision making over time.
