## Product

System for creating rollups on Bitcoin, using something called "BitSNARKs". This
makes it possible to bridge BTC trustlessly to other chains, using ZK proofs.
The proofs are not executed on-chain, but through external actors reading and
writing Bitcoin UTXOs. Somewhat similar to how the Lightning Network state
channels work.

So essentially: execute off-chain, prove results, settle on-chain.

Then there is the entire system that performs all this: BitcoinOS. Which
basically provides a bridge and an execution environment for Bitcoin rollups, to
use bridged BTC in turing complete computation. I think? I don't understand
exactly how it works.

Raised $10M in October 2025.
