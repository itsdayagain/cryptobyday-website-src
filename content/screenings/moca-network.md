## Product

Identity infra.

Main products are the Moca Chain, an EVM-compat L1 that stores user info, and
the SDK "AIR Kit" that provides developers with functionality for user identity,
assets, reputation. Uses ZK tech for privacy.

Serves as the identity layer for Animoca Brands.

Moca Chain mainnet to be launched Q4 2025.

## Token

Token $MOCA is the native token of the Moca Chain. Used for gas, various fees,
governance.

The token is older than the Moca Chain though. It was originally launched on
Ethereum, and is now also on Base.
