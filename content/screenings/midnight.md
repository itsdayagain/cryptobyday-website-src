## Product

Privacy focused L1, using ZK Snarks. New smart contract language "Compact",
based on TypeScript.

UTXO-based. An account-based smart contract system on top. Two types of tokens:
ledger tokens (UTXO) and smart contract tokens (account based).

Still in testnet as of December 2025. Token launched on 9 Dec 2025. Mainnet
planned for launch in 2026.

Website and docs all look very professional.

## Token

Token \$NIGHT is the native token. Also a \$DUST token, that is generated by
\$NIGHT, which is used to pay gas, and also it's actually not a token.

## Team

Charles Hoskinson is involved. So the hype machine should be on.

Midnight Foundation: organization "dedicated to growing the Midnight network".

Shielded: engineering company. Spinout from Input Output. So probably good on
the crypto side, likely not on high performance engineering.
